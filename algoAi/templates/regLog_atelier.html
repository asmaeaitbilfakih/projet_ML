Atelier Pratique - Classification des véhicules

Dataset des véhicules

L'objectif de ce démonstration est de vous montrer comment on entraîne un modèle de régression logistique pour classifier les véhicules en Camion ou Touristique selon le nombre de roes et la hauteur donné

Les modèles entraînés et sauvegardés de cette atelier vont être utiliser pour faire des classification à base des données qui vont être saisir par l'utilisateur de l'application via un formulaire une fois il clique sur le bouton "tester le modèle
