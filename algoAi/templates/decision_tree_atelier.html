Atelier Pratique - Arbre De Decision

Dataset des locations

L'objectif de cette démonstration est de montrer comment entraîner un modèle pour prédire la recommandation d'une location selon ses caractéristiques :

  • Latitude et Longitude
  • Capacité d'accueil
  • Distance
  • Nombre de chambres
  • Distance au métro
  • Satisfaction des invités
  • Attractivité

Les modèles entraînés seront utilisés pour prédire la recommandation (1 = recommandé, 0 = non recommandé) à partir des données saisies par l'utilisateur via un formulaire.
